## Quantidade de Lunes

A quantidade necessária para Validação de blocos e ganhar os Tokens é de **5000 Lunes**.

## Proof-of-Stake

Nesse consenso de algoritmo, o node responsável por validar o bloco é gerado aleatóriamente.
No entanto, o node deve ter **estocado**(stake) **5000 Lunes** ou mais para poder validar um bloco.
Quanto mais lunes você tiver estocado(stake), maiores a chances de você ser escolhido para validar um bloco e receber a recompensa.
É tambem possível receber Lunes para seu node através de **Stake**, e acrescentar suas chances de ganho.
Para enconrajar os usuários a fazerem Stake você pode oferecer recompensas, tokens, etc...

Quando as lunes são emprestadas(stake), elas **não são transferidas**, elas estão apenas trancadas na sua própria carteira.
**O pagamento das recompensas são de inteira responsabilidade do administrador do node, NÃO DO MECANISMO DA BLOCKCHAIN**.

### Processo de Stake

Stake é o processo de emprestar suas lunes para um administrador de um full node que tem mais de 5000 lunes
e receber recompensas, previamente concordadas por ele, podendo ser tokens, lunes, points etc...
