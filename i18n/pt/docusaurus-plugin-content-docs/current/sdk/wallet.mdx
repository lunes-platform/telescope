# 🔐 Carteira

Uma **Carteira** é um conjunto de uma ou mais **Contas**, é como se fosse uma conta de email
só que em vez de ter um _endereço@provedor_ e  _senha_ a carteira tem:

- seed: uma lista de palavras aleatórias usada para gerar chaves e endereço
- chain: um Fork da blockchain, que pode ser mainnet ou testnet
- nonce: número usado para gerar novas chaves com a mesma seed
- chave privada: chave usada para assinar transações
- chave publica: chave derivada da private key usado para derivar o endereço
- endereço: endereço publico para receber ativos

A conta pode ser gerada na testnet ou mainnet. **Isso vai apenas mudar o endereço**.
A seed usada para criar a conta contém uma lista de 12 ou mais palavras,
é possivel gerar uma seed de até 2048 palavras.
A chance para duas contas serem criadas com as mesmas palavras é de $\frac 1 {2048^n}$ onde $n$ é o número de palavras.
VocÊ pode gerar uma conta passando a sua seed, sua chave privada, chave publica e o endereço como **parâmetro** ou simplesmente não passando nada.
Mudando o nonce é possivel gerar 4 294 967 295 diferentes contas apenas com uma seed.

## Criando uma nova carteira a partir da _seed_, _nonce_ e _chain_

<Tabs>
  <TabItem value="Python3" label="Python3" default>

```py
from lunespy.client.wallet import Account

wallet = Account(
    seed = "phone dove naive find erase grant right cause garden struggle robust ball",
    nonce = 0,
    chain = "mainnet"
)

print(wallet)
```
