# 🏁 Empezando

## Introducción

El **Lunes SDK ** acerca a los desarrolladores a la blockchain de Lunes, proporcionando un conjunto completo de funciones que necesitan las aplicaciones descentralizadas (DApp) o la aplicación Blockchain. Ayuda a los desarrolladores a administrar cuentas de blockchain fácilmente y facilita las transacciones.

La biblioteca está impulsada por transacciones en la arquitectura _lunesnode_.
**El concepto de transacción aquí será cualquier tipo de interacción con la blockchain de lunes.**
De esta forma, podemos crear una transacción para transferir tokens, una transacción para crear tokens, una transacción para crear un nickname (alias) para nuestra cuenta, etc.
