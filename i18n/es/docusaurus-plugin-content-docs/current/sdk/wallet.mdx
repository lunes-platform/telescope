# 🔐 Wallet

Una **Wallet** es un conjunto de una o más **Cuentas**, como una cuenta de correo electrónico, que en vez de tener _address@provider_ and _password_ has:

- seed: lista de palabras aleatorias utilizadas para generar claves y direcciones
- chain: bifurcación blockchain, puede ser mainnet o testnet
- nonce: número utilizado para generar nuevas claves con la misma semilla
- private key: clave utilizada para firmar transacciones
- public key: clave derivada de la clave privada utilizada para derivar la dirección
- address: dirección pública utilizada para recibir activos

La cuenta se puede generar para testnet o mainnet. **Esto solo cambiará el address**,
La seed utilizada para crear una cuenta debe ser una lista de 12 o más palabras,
es posible generar una semilla con hasta 2048 palabras.
La probabilidad de que se creen dos cuentas iguales es $\frac 1 {2048^n}$ donde $n$ es el número de palabras.
Puede generar una cuenta pasando su seed, private_key, publick_key y address como **parámetros** o nada para generar una nueva.
Cambiando el nonce es posible generar 4 294 967 295 cuentas diferentes con una sola seed.

## Crear el Wallet de _seed_, _nonce_ and _chain_

<Tabs>
  <TabItem value="Python3" label="Python3" default>

```py
from lunespy.client.wallet import Account

wallet = Account(
    seed = "phone dove naive find erase grant right cause garden struggle robust ball",
    nonce = 0,
    chain = "mainnet"
)

print(wallet)
```
