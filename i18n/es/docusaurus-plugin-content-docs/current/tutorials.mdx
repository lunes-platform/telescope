## C贸mo funciona Lunes Node

Lunes Blockchain es una red descentralizada y distribuida basada en el algoritmo de consenso LPoS. Esta red puede ser mainnet o testnet, en mainnet los nodos funcionan con valores reales de tokens, monedas, NFT y activos. Por otro lado, testnet tiene tokens "falsos" para proporcionar un entorno de prueba.
En el lenguaje de desarrollo, mainnet es prod y testnet es dev.

Lunes Node es un node o servidor que tiene la responsabilidad de mantener y validar la blockchain, transmitir nuevas transacciones y proporcionar una API para la interacci贸n e integraci贸n con la red.

Con un Lunes Node, se convierte en un participante de Lunes Blockchain y puede proporcionar resistencia a la red, validar datos e integrar servicios del mundo real en la blockchain.
