## Create NFT

  ```python
  from lunespy.client.transaction.issue import IssueNFT

  tx = IssueNFT(
    description="Tokenizing my Ford Mustang Shelby GT500 1967",
    reissueable=False,
    name="Eleanor",
    sender=wallet,
    quantity=1,
    decimals=0
  ).send()
  ```
