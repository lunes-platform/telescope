## Full Example

<Tabs>
  <TabItem value="Python3" label="Python3">

  ```python
  from lunespy.wallet import wallet_factory

  wallet = wallet_factory(
      seed="abc", # you can create a seed with any string
      private_key="", # only if it doesn't pass the seed
      chain=1, # 1 for mainnet, 0 fo testnet
      nonce=0, # 0 by default, but can go up to 
      seed_len=15, # number of words of the seed, only when creating a new seed
  )
  ```
