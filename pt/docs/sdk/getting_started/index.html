🏁 Começando

Introdução​

O Lunes SDK traś os desenvolvedores pra mais perto da Lunes Blockchain, providenciando uma série de aplicativos descentralizados(DApps) ou aplicações que a Blockchain necessita. Isso ajuda os desenvolvedores a gerenciar suas contas da Blockchain e fazer transações de formar mais prática.

A Biblioteca entrega as transações na arquitetura lunesnode. O conceito de transação aqui é qualquer tipo de interação com a blockchain da Lunes. Diante disso, nós podemos criar transações para transferência de tokens, uma transação para a criação de tokens, uma transação para criar um apelido(alias) para sua carteira, etc.

Como Instalar​

  • Python3
  • JavaScript
  • Rust
pip install lunespy
npm i lunescrab
